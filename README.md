# GDUE-SMS 学生管理系统

一个基于前后端分离架构的学生信息管理系统，采用 Spring Boot + Vue 3 技术栈开发。

## 项目简介

GDUE-SMS（学生管理系统）是一个现代化的Web应用，用于管理学生基本信息和专业信息。系统提供完整的CRUD功能，支持分页查询、条件筛选等特性。

## 技术栈

### 后端
- Java 25
- Spring Boot 3.5.9
- MyBatis 3.0.5
- MySQL 8.0+
- PageHelper 1.4.7（分页插件）
- Lombok

### 前端
- Vue 3.2.38
- Vue Router 4.1.5
- Element Plus 2.4.4
- Axios 1.7.2
- Vite 7.2.4
